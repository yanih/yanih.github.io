学习回顾·第零次

黄雅妮 \ 2020年9月结束

去年十一，得师兄警告，开学第一个月老板们会组织过去一年的学习汇报。想想，从去年暑假到现在三月，我也是没少浪费时间（抹泪），“真正的勇士敢于直面惨淡的人生”，不如就从现在开始积累吧。

2019年9月-2020年3月5日

理论和文献

✔︎《概率论与数理统计》-mooc：了解概率分布背后的定律和推导。
✔︎ 大话统计系列讲座-阜外/协和：了解科研实际用到的统计学知识。
○ 毕设-在家：练习读文献，英文论文写作。
总结：多读书。

Python

○ Python官方文档，非常好。 ○ 《零基础入门学习python》-b站小甲鱼，优点是能够每课上鱼C论坛写题练代码,非常好的实践操作，缺点是这些练习要钱。
○ Python for everybody-coursera：优点是Charles教授讲课生动有趣，有任务式的代码题可做，缺点是每课知识点较少，题偏简单，相对有些浪费时间。
○ Python-100-Days：github，优点是知识点串讲，有大量经典题型，缺点是新手偏难，需要建立一定基础，再来花时间好好琢磨。
总结：多练习。

R

✔︎ Data science: Foundations using R-coursera JHU：入门安装。
○ An Introduction to Statistical Learning：电子书学习。
总结：目前与R没有姻缘，再说吧。
