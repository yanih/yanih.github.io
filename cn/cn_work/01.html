一年学习回顾·第零次

去年十一，得师兄警告，开学第一个月老板们会组织过去一年的学习汇报。想想，从去年暑假到现在三月，我也是没少浪费时间（抹泪），“真正的勇士敢于直面惨淡的人生”，不如就从现在开始积累吧。我的学习轴心落在统计学理论和数据分析软件（Python和R）上。

2019年7月-2020年3月

理论和文献

✔︎《概率论与数理统计》-mooc：收获还可，了解概率分布背后的定律和推导，有笔记可复习。
✔︎ 大话统计系列讲座-现场：了解科研实际用到的统计学知识，有笔记可复习。
○ 毕设（T2D·biomarker·metabolomics）-在家：第一次英文论文写作，全靠读文献摸索（拼凑）。 总结：统计学知识太少，多读书。

Python

○ 《零基础入门学习python》-b站小甲鱼，写笔记做题。
○ Python for everybody-coursera：喜欢Charles！
○ Python-100-Days：github，用来巩固知识。 总结：多练习。

R

✔︎ Data science: Foundations using R-coursera JHU：只学了课程1，安装R和Rstudio.
○ An Introduction to Statistical Learning：读电子书，用R练习，暂时搁置。

总结：再说吧。
